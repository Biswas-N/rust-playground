# Rust Playground

Playground is a Rust based CLI-application and a reference notes based on [Rust Crash Course](https://www.youtube.com/watch?v=zF34dRivLOw) by [Traversy Media](https://www.youtube.com/channel/UC29ju8bIPH5as8OGnQzwJyA)

## Installation

Download(Clone) the repository and just run the code below.

```bash
cargo build --release 
./target/release/playground --help 
```

## Usage

```bash
./target/release/playground -c Functions
```
Run the above command (Case-insensitive) and head to file to view detailed explanation of the code.
